# OpenClaw Docs — Stored Versions

Each version has three artifacts:

- `openclaw-docs.<version>.md` — flattened docs
- `openclaw-docs.<version>.toc.jsonl` — TOC (sections + H2/H3 + keywords + line ranges)
- `openclaw-docs.<version>.sections.jsonl` — section offsets (line + byte ranges)

`openclaw-docs.latest.*` are real copies of the highest-version files.
